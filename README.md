# Tic-Tac-Toe (Rust)

A two-player Tic-Tac-Toe game for the terminal, written in Rust.

## How to Run

```bash
cargo run
```

## How to Play

- Two players take turns: **X** goes first, then **O**
- On your turn, enter your move as `row col` (space-separated, 0-indexed)

- 0 0  →  top-left
1 1  →  center
2 2  →  bottom-right

- The game ends when a player wins or the board fills up (draw)
